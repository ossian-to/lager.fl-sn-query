使用方式

第一步：載入 popup.js
第二步：載入樣式檔 popup.css
第三步：載入Popup 的 HTML Tag

如何呼叫 popup?

在需要呼叫 popup 的地方增加 popup-trigger 的 class
EX: <a href="#" class="popup-trigger">Show Popup</a>

如何關閉 popup

在需要關閉 popup 的地方增加 popup-close 的 class
EX: <button class="popup-close">Close</button>
